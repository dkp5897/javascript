AJAX stands for asynchronous javascrip and XML. 
AJAX is not a programming language, rether, it is a set of existing technologies.
AJAX helps in fetching data asynchronously wthout interfering with the existing page. 
No page reload/refresh
Modern websites use JSON instead of XML for data transfer.


Why use AJAX ? 

No page reload/refresh
Better user experience
Saves network bandwith
Very Interective
